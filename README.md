## вҡҷпёҸ Installation

### Prerequisites
- Node.js 18+
- A [Groq API key](https://console.groq.com/) (free)
- A [Supabase](https://supabase.com/) project (free)

### 1. Clone the repo
```bash
git clone https://github.com/Prajwal18py/codevis.git
cd codevis
```

### 2. Install dependencies
```bash
npm install
```

### 3. Set up environment variables
```bash
cp .env.example .env
```
Fill in your `.env`:
```env
VITE_GROQ_API_KEY=your_groq_api_key_here
VITE_SUPABASE_URL=https://yourproject.supabase.co
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key_here
```

### 4. Set up Supabase database
Run this in your **Supabase SQL Editor**:
```sql
create table saved_results (
  id uuid primary key default gen_random_uuid(),
  user_id uuid references auth.users(id) on delete cascade,
  type text not null, title text not null, lang text default 'cpp',
  result jsonb not null, created_at timestamptz default now()
);
create table quiz_scores (
  id uuid primary key default gen_random_uuid(),
  user_id uuid references auth.users(id) on delete cascade,
  topic text not null, score int not null, total int not null,
  pct int not null, created_at timestamptz default now()
);
create table streaks (
  id uuid primary key default gen_random_uuid(),
  user_id uuid references auth.users(id) on delete cascade unique,
  current int not null default 0, longest int not null default 0,
  last_date date not null, updated_at timestamptz default now()
);
```

### 5. Run locally
```bash
npm run dev
# Open http://localhost:5173
```

## рҹҡҖ Deploy to Vercel

```bash
git add .
git commit -m "рҹҡҖ CODEVIS v4 вҖ” AI DSA Platform"
git push origin main
```

1. Go to [vercel.com](https://vercel.com) вҶ’ **Add New Project** вҶ’ Import `codevis`
2. Add environment variables:
   - `VITE_GROQ_API_KEY`
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
3. Click **Deploy** вң…
4. Add Vercel URL to **Supabase вҶ’ Auth вҶ’ Redirect URLs**:
